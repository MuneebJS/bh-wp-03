== Description ==

Insynia is a modern full-site editing theme. Its block templates, template parts, patterns, fonts, and images are carried over from Twenty Twenty-Five; global colors and typography presets are retuned for a contemporary look. Front-end styling and scripts are maintained as plain files under assets/css/insynia.css and assets/js/insynia.js (no build pipeline).
